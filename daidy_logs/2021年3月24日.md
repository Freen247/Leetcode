### golang的协程goroutine
Goroutine 是与其他函数或方法同时运行的函数或方法。 Goroutines 可以被认为是轻量级的线程。 与线程相比，创建 Goroutine 的开销很小。 Go应用程序同时运行数千个 Goroutine 是非常常见的做法。

- 指针的作用？
	```
	指针用来保存变量的地址。
	var x =  5
	var p *int = &x
	fmt.Printf("x = %d",  *p) // x 可以用 *p 访问
	* 运算符，也称为解引用运算符，用于访问地址中的值。
	＆运算符，也称为地址运算符，用于返回变量的地址。
	```
