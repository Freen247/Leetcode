|\||按位或|a | b|4 \| 5 == 0100 \| 0101 -> 0101 |
|^|按位异或|a ^ b|4 ^ 5 == 0100 ^ 0101 -> 0001 |
|~|按位取反|~a|~4 == ~0100 -> ~ 1011|

|>>|按位右移|a >> b|4 >> 2，表示整数 4 按位右移 2 位|
